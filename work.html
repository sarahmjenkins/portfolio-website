Meet App

This project connects to CareerFoundry's calendar API to show upcoming events all over the world. Users are able to use Google OAuth to access the app; sort events by city, change the number of events rendered, expand and hide a detailed view of events, and view charts showing the number of events per city and distribution of topics of events. This project is a serverless, progressive web application using React and a test-driven development approach. Created in October 2022.
